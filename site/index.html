Counter

Counter Challenge Requirements

  • Build a frame containing a display D and a button B.
  • Initially, the value in D is 0 and each click of B increases the value in D by one.

My Solution

Although the requirements ask for only two elements (a display field and a button to increment) I wanted more than one option for users to be able to interact with the GUI.

My additional features include:

  • An option to decrement the value.
  • Accessibility improvements such as ARIA labels, and key press listeners for users that rely on screen readers and keyboard navigation.
    • Try turning your screen reader on , move your focus to the counter, and then use the Up and Down Arrow keys.
